position的值

static

static为默认值，指示元素出现在正常的文档流中；块级元素生成一个矩形框，作为文档流的一部分，行内元素则会创建一个或多个行框，置于其父元素中

relative

relative相对定位用法一:元素相对自身的原位置进行偏移，但是原本的空间依旧保留，表现为空白。 用法二:把一个元素设置为position:relative;可以使该元素的子元素相对该元素绝对定位。

absolute

absolute绝对定位 元素框从文档流完全删除，并相对于其包含块定位。元素原先在正常文档流中所占的空间会关闭，就好像元素原来不存在一样。

fixed

fixed 固定定位 元素从文档流删除，并相对于浏览器视窗定位，因此不随文档滚动而移动。元素在原本的空间关闭。

sticky

sticky 粘性定位 是相对定位和固定定位的混合。元素在跨越特定阈值前为相对定位，之后为固定定位。关于这个属性的详细信息请参考这个链接 More info: sticky属性

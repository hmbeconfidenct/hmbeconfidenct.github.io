CSS-Position

position的值

static

static为默认值，指示元素出现在正常的文档流中；块级元素生成一个矩形框，作为文档流的一部分，行内元素则会创建一个或多个行框，置于其父元素中

relative

relative相对定位用法一:元素相对自身的原位置进行偏移，但是原本的空间依旧保留，表现为空白。 用法二:把一个元素设置为position:relative;可以使该元素的子元素相对该元素绝对定位。

absolute

absolute绝对定位 元素框从文档流完全删除，并相对于其包含块定位。元素原先在正常文档流中所占的空间会关闭，就好像元素原来不存在一样。

fixed

fixed 固定定位 元素从文档流删除，并相对于浏览器视窗定位，因此不随文档滚动而移动。元素在原本的空间关闭。

sticky

sticky 粘性定位 是相对定位和固定定位的混合。元素在跨越特定阈值前为相对定位，之后为固定定位。关于这个属性的详细信息请参考这个链接 More info: sticky属性

CSS Base

CSS优先级

共分为4个等级：
• 第一等：代表内联样式，如: style=”xxx”，权值为1000。
• 第二等：代表ID选择器，如：#content，权值为0100。
• 第三等：代表类，伪类和属性选择器，如.content，:hover，[attribute]，权值为0010。
• 第四等：代表元素选择器和伪元素选择器，如div，p，权值为0001。

CSS伪类

一个 CSS 伪类（pseudo-class)是一个以冒号(:)作为前缀，被添加到一个选择器末尾的关键字，当你希望样式在特定状态下才被呈现到指定的元素时，你可以往元素的选择器后面加上对应的伪类（pseudo-class）。你可能希望某个元素在处于某种状态下呈现另一种样式，例如当鼠标悬停在元素上面时，或者当一个复选框被禁用或被勾选时，又或者当一个元素是它在 DOM 树中父元素的第一个孩子元素时。

CSS伪元素

伪元素（Pseudo-element）跟伪类很像，但它们又有不同的地方。它们都是关键字，但这次伪元素前缀是两个冒号 (::) ， 同样是添加到选择器后面去选择某个元素的某个部分
